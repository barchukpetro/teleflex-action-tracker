Firebase Setup Required for Multi-User Sync

The tracker works locally right now. To enable live sync across multiple users, complete these steps:

  1. Go to console.firebase.google.com and create a free project
  2. Click the web app icon (</>) to register an app and copy the config
  3. Go to Realtime Database → Create Database (choose any region, start in test mode)
  4. Open this HTML file in a text editor and paste your config values into FIREBASE_CONFIG
  5. Host this file on GitHub Pages, Netlify, or any static host — then share the URL
